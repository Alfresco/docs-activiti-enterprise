# Deploying with custom images
It is possible to replace the Activiti Enterprise images for certain services with custom images. The services that can be replaced are the following:

* Audit service
* Content service
* DMN service
* Identity service
* Form service
* Messaging service
* PostgreSQL
* Preference service
* Query service
* Runtime Bundle
